# 美伊局势日报来源优先级

本部分属于《美伊局势日报》专用，不可替代 `shared/source-policy.md`。

## 一手与官方来源
1. **美军与联军方**：US CENTCOM (美国中央司令部) 每日战报（击落无人机/反舰导弹数量）、US DoD (五角大楼)、UKMTO (英国海上贸易行动办公室 - 负责红海商船遇袭通报)。
2. **伊朗方**：伊朗外交部、IRGC、伊朗国家通讯社 (IRNA)。

## 国际/专业媒体
- 全球通讯社：Reuters, AP, Bloomberg.
- 航运与能源专业来源：Lloyd's List, S&P Global Platts, 追踪航运改道和油价异动。

## 智库与开源情报
- 追踪美军基地被袭与红海局势的专业 OSINT 账号与防务智库。
